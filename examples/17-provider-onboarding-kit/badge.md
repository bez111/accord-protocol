# Accord Compatibility Badge Language

Use one of these only after publishing the matching evidence.

```markdown
[![Accord-compatible: L0](https://img.shields.io/badge/Accord--compatible-L0-blue)](https://accordprotocol.ai/learn/conformance-levels/)
```

```markdown
[![Accord-compatible: L1 testnet](https://img.shields.io/badge/Accord--compatible-L1%20testnet-blue)](https://accordprotocol.ai/learn/conformance-levels/)
```

Required nearby text:

```text
Accord-compatible means this provider emits Accord v0 protocol objects and has
published conformance evidence. It does not imply mainnet certification,
financial safety, or audit completion.
```

Do not use "mainnet", "production", "audited", or "certified" unless signed
external audit manifests explicitly allow the exact artifacts you use.
