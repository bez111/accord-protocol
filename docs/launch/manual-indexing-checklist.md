# Manual Indexing And Launch Checklist

These tasks require project-owner accounts or community accounts. They are intentionally not automated from CI.

## Search indexing

- [ ] Add `accordprotocol.ai` to Google Search Console.
- [ ] Submit `https://accordprotocol.ai/sitemap.xml` in Google Search Console.
- [ ] Add `accordprotocol.ai` to Bing Webmaster Tools.
- [ ] Submit `https://accordprotocol.ai/sitemap.xml` in Bing Webmaster Tools.
- [ ] Request indexing for `https://accordprotocol.ai/`.
- [ ] Request indexing for `https://accordprotocol.ai/learn/`.

- [ ] Request indexing for `https://accordprotocol.ai/learn/agent-payments-stack/`.
- [ ] Request indexing for `https://accordprotocol.ai/learn/paid-mcp-tools-production-checklist/`.

## Launch posts

- [ ] Publish Show HN using `docs/launch/hn-accord-launch.md`.
- [ ] Publish one adapted Reddit post using `docs/launch/reddit-accord-launch.md`.
- [ ] Publish X thread using `docs/launch/x-accord-thread.md`.
- [ ] Post the MCP community variant from `docs/launch/discord-accord-announcement.md`.
- [ ] Post the x402/payment-infrastructure variant from `docs/launch/discord-accord-announcement.md`.
- [ ] Post the Ergo/rail community variant from `docs/launch/discord-accord-announcement.md`.
- [ ] Submit or refresh the mcp.so listing using `docs/launch/mcp-so-listing.md`.

## Follow-up window

- [ ] Reply to HN/Reddit/community comments within 24 hours.
- [ ] Capture useful questions as GitHub issues or Learn article follow-ups.
- [ ] Check Search Console coverage after 48-72 hours.
- [ ] Check `site:accordprotocol.ai accord protocol` after 48-72 hours.

## Required wording

Keep this sentence in public launch replies where status is discussed:

> Accord Protocol v0 is alpha / testnet-first and NOT CERTIFIED FOR MAINNET.

Do not imply production adoption, external audit completion, mainnet safety, custody, or real-fund production readiness.
