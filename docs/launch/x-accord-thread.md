# X thread draft: Accord Protocol

Do not post until the canonical website, GitHub README and public status page are live.

## Tweet 1

We launched Accord Protocol: an open standard for autonomous agent work agreements.

x402 verifies payment.
Accord verifies completion.

Website: https://accordprotocol.ai/
Repo: https://github.com/accord-protocol/accord-protocol

## Tweet 2

Paid agent workflows need more than a payment proof.

They need a record of:
- what work was promised
- how payment authority was expressed
- who verified the output
- whether settlement closed

Accord records that lifecycle as protocol objects.

## Tweet 3

The core objects:

1. Accord Agreement
2. Verification Receipt
3. Settlement Receipt

Learn the model:
https://accordprotocol.ai/learn/agent-work-agreements/

Agreement object:

## Tweet 4

Accord is designed to compose with adjacent layers:

- MCP: how agents call tools
- x402: paid HTTP resources
- AP2-style flows: payment authority
- Accord: work terms, verification and settlement receipts

Comparison guides:
