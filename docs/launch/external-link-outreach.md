# External Link Outreach Targets

The goal is not spam. Each external link should help a real developer understand Accord and should point to a canonical, useful page on `accordprotocol.ai`.

## Links already controlled by this repo

1. GitHub repository README
   - Target URL: https://accordprotocol.ai/
   - Anchor/copy: Website, Learn hub, Status, Security, Roadmap, Sitemap.

2. Package READMEs
   - Target URLs:
     - https://accordprotocol.ai/learn/accord-mcp-paid-tools/
     - https://accordprotocol.ai/learn/accord-402-flow/
     - https://accordprotocol.ai/learn/conformance-levels/
     - https://accordprotocol.ai/status/
   - Purpose: make npm/GitHub package pages point back to public docs.

3. GitHub repo metadata homepage
   - Target URL: https://accordprotocol.ai/

4. GitHub Release for the current package line

## Posting rules

- Keep the phrase "alpha / testnet-first and NOT CERTIFIED FOR MAINNET" when discussing current status.
- Do not imply production adoption, external audit completion or mainnet safety.
- Prefer deep links over homepage-only links when the community has a specific interest.
- Reply to questions within 24 hours; dead launch threads waste the backlink.
